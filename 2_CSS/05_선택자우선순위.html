CSS 선택자의 우선순위

        기본적으로 CSS속성은 STYLE태그 또는 CSS파일에 작성된 순서되로 해석이 진행된다.
        
        같은 요소(태그)에 여러 CSS속성이 설정되는 경우 "우선순위"가 적용된다.

        1) 동일한 우선순위로 CSS속성이 설정된 경우 "가장 마지막에 작성된" CSS 속성이 반영된다.
        2) 여러 선택자를 이용해서 같은 요소를 선택하더라도 지정되는 CSS 속성이 다르면 모두 반영된다.
        
        0순위 : css속성 : 속성값 !important;
        1순위 : inline-style속성(요소에 직접 지정되는 style속성)
        2순위 : 아이디 선택자 (#아이디 속성명)
        3순위 : 클래스 선택자 (.클래스 속성명)
        4순위 : 태그 선택자(태그명)
