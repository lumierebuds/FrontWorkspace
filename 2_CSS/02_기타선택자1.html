1. 속성 선택자

스타일을 부여하고자 하는 요소 내에 작성되어 있는 속성을 이용해서
선택하는 방법. 선택자 뒤에 []를 이용해서 속성과 속성값을 제시. 

        기본선택자[속성=속성값]
        기본선택자[속성~=속성값]
        기본선택자[속성|=속성값]
        기본선택자[속성^=속성값]
        기본선택자[속성$=속성값]
        기본선택자[속성*=속성값]

2. 자손선택자와 후손선택자

자손 : 바로 하위인 요소들 후손 : 모든 하위요소 

        * 자손선택자 
        a>b{
            스타일 속성
        }

        * 후손선택자 
        a b{
            스타일 속성
        }

3. 동위(같은레벨) 선택자

동위관계에 있는 요소의 뒤에 위치한 특정 요소를 선택할때 사용 

        * a요소 뒤에 있는 b요소 "하나만"을 선택
        a+b{
            스타일
        }

        * a요소 뒤에 있는 "모든" 요소를 다 선택 
        a~b{
            스타일
        }

4. 반응선택자

사용자의 움직임에 따라 선택이 되거나 안되는 선택자 

        * 해당 요소에 "클릭"이 되었을 경우 반영되는 스타일
        선택자:active{
            스타일 속성: 값
        }

        * 해당 요소에 마우스가 "올라가는" 순간 반영되는 스타일
        선택자:hover{
            스타일;
        }

5. 상태 선택자

요소의 상태에 따라 선택되는 선택자 

        1) checked된 상태의 요소
        선택자:checked{
            스타일
        }

    
        2) 초점(focus)이 맞춰진 input 요소 선택 
        선택자:focus{
            스타일
        }
