레이아웃(layout)

        사전적 의미 : 배치, 정리
        기술적 의미 : 구성요소를 제한된 공간에 효율적으로 배치하는것을 의미

화면 배치 방법 : display 속성

        요소가 화면에 어떻게 보여질지 형식을 지정하는 속성

        - block : 화면을 수직분할(행을 나눔)
                  width / height 속성 사용 가능.

        - inline : 화면을 수평분할(하나의 행의 칼럼을 나눔)
                   width / height 속성 사용 불가능.
        - inline-block : inline의 수평분할 + block의 크기 조정

        - none : 화면에 요소가 표시되지 않으나 존재는 하고있는 상태로 지정
