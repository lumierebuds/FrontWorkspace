border

        * 테두리 스타일
        border-방향-style : none/dotted/dashed/solid/double/groove/ridge/inset/outset;

        * 테두리 두께 
        border-방향-width : 테두리 두께; 

        * 테두리 색상
        border-방향-color : 테두리 색상;

        * 한번에 지정 
        border-방향 : 두께 스타일 색상;

        border-방향-radius : 숫자; => 테두리를 둥그스름하게

        * 박스에 그림자 효과 
        box-shadow : 가로거리(x) 세로거리(y) 흐림정도 번짐정도 색상;
