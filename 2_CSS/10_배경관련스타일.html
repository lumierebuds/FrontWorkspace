배경 이미지 테스트

        선택자{
            background-image : url(이미지경로);
            background-repeat : 배경이미지의 x/y축 반복
            background-position : 배경이미지의 위치 
            background-size: contain(x/y축중 한쪽이 꽉찰때까지 확대/축소) 
                             , cover(모두 채울수 있도록 확대/축소)
            background-attachement : scroll(페이지스크롤시 영역과 함께 스크롤)
                                     fixed(영역과 별개로 화면에 고정)
            background-origin : 배경이미지 시작 위치 
        }
