CSS(Cascading Style Sheets)

        마크업 언어(HTML)가 실제로 화면에 
        표시되는 방법들을 기술하는 언어. 
        - HTML코드가 화면에 어떤 스타일로 보여질지 지정하는 언어.
        - 
        
        Cascading? 
        - 연속적인 스타일 상속
        - 상위요소에 적용한 스타일이 마지막 하위요소까지
          연속적으로 전파(상속)됨을 의미.

CSS 기본 선택자

CSS 선택자의 종류

1. 모든(전체) 선택자 (*)

모든 요소들을 다 선택하고자 할때 사용 

        * {
            스타일 속성 : 값; 
            스타일 속성 : 값; 
            ....
        }

2. 태그 선택자 (태그명)

문서내에 같은 태그를 모두 선택할때 사용 

        태그명 {
            스타일 속성: 값; 
        }

3. 클래스 선택자(.클래스명)

현재 문서 내에서 원하는 여러개의 요소를 선택하고자 할 때 사용한다. 

        .클래스명 {
            스타일
        }

4. 아이디선택자 (#아이디명)

문서내에 특정 html요소 하나만을 선택하고자 할 때 사용 단, 그 요소가 id속성을 통해 고유한 id값이 있어야함. 

        #아이디명{
            스타일
        }
