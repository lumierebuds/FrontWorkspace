목록 관련 스타일(list~)

list-style-type : 불릿기호를 변경할 수 있는 스타일

        * 순서없는 목록(ul)
            list-style-type : disc/circle/square/none;
        * 순서 있는 목록(ol)
            list-stype-type : decimal/decimal-leading-zero;
                              lower-alpha/upper-alpha;
                              lower-roman/upper-roman;

순서 없는 목록

  • HTML
  • CSS
  • JS
  • JQUERY

list-style-position : 불릿기호의 위치를 조정할때 사용

list-style-image : 불릿기호로 이미지를 적용할때 사용

        list-style-position : inside/outside;
        list-style-image : url(적용시키고자 하는 이미지 경로);
